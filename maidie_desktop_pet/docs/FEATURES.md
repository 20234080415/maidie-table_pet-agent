# 功能说明

## 桌宠窗口

- PyQt6 无边框透明窗口，默认始终置顶。
- 支持拖动、边缘缩放、滚轮缩放和屏幕边界约束。
- 聊天气泡与输入框是独立跟随浮窗，会避开角色和屏幕边缘。

## 动画系统

主图集负责 `idle`、行走、奔跑、思考、说话、等待、成功和失败等状态。额外动作使用 `assets/actions/` 下的数据驱动 WebP 动作条，可配置帧率、冷却和优先级。详见[动作系统](ACTIONS.md)。

## 鼠标和键盘互动

- 点击头顶或左右抚摸：摸头反应。
- 点击脸颊：戳脸反应。
- 点击身体：自然互动。
- 明显拖动：移动 Maidie；特定拖动手势可触发动作。
- 双击角色或按 Enter / 空格：打开输入框。
- Esc、失焦、发送完成或超时后收起输入框。

## 围栏模式

右键菜单可启用围栏。围栏支持移动和缩放，并统一约束自主游走、拖拽与释放后的回弹位置。`fence.show_overlay` 只控制边框显示，不关闭范围约束。

## AI 聊天

- 支持 DeepSeek 和 OpenAI Chat Completions 兼容接口。
- SSE 流式内容经过分句和节奏控制后显示。
- 普通聊天不应调用屏幕或事实工具；技术、事实和桌面任务进入对应 Agent 路径。

## 人格与记忆

- 内置温柔傲娇、元气活泼、安静治愈、优雅女仆和自定义人格。
- 近期对话、长期事实和偏好保存在本地 SQLite。
- 敏感信息会经过规则过滤；可从右键菜单清除记忆。

## Agent 工具能力

当前生产工具包括：

| 工具 | 能力 |
|---|---|
| `TimeTool` | 当前时间、日期和事件倒计时 |
| `WeatherTool` | 当前或次日天气事实 |
| `SearchTool` | Tavily 搜索摘要与来源 |
| `ScreenTool` | 按指定范围取得结构化屏幕事实 |
| `MemoryTool` | 查询本地近期对话、事实与偏好 |
| `SystemTool` | 受控文件读取、搜索和需确认的有限操作 |

工具只提供事实，最终文本由 Synthesizer 生成。

## 时间与倒计时

Router 能区分当前时间与目标时间差，例如：

```text
现在几点了
到17:40还有多久
我5.40下课，现在还有多久下课
```

支持 `5.40`、`5:40`、`17:40`、`5点40`、`下午5点40`、`晚上8点` 等表达。目标时间已过时会返回明确状态，不制造错误倒计时。

当前会话可暂存事件时间。先说“我5.40下课”，之后可问：

```text
还有多久下课
```

该上下文不写入长期记忆，应用重启后清除。

## 天气

天气工具读取结构化天气数据并带有短缓存。城市识别和建议类问题仍受当前服务覆盖范围及模型路由质量限制，不应把模型推测当作实时天气。

## 搜索

联网查询默认关闭。启用并配置 Tavily 后，可处理明确搜索请求及安全的连续查询。网络错误、超时、缺少 Key 或空结果会返回结构化失败信息。

## 屏幕理解

- 明确请求时才调用 `qwen3-vl-flash`。
- 支持当前窗口、全屏、鼠标附近和手动框选。
- 模糊的“帮我看看”会先确认，不会立即截图。
- 视觉模型只输出结构化事实；最终回答由 Synthesizer 生成。
- 本地 OCR 与按需 Qwen VL 是两条独立链路。

详见[千问视觉与屏幕理解](VISION.md)。

## 主动行为

主动行为默认关闭。启用后可根据空闲、长时间编程、场景变化、窗口切换或剪贴板变化生成有限提示，并受全局冷却控制。主动引擎只产出行为决策，不另建 AI 回答链路。

## 桌面感知

可用上下文包括鼠标活动、空闲时长、前台窗口、应用类型、窗口变化和可选 OCR。剪贴板默认只检测是否变化，不读取正文。

## 状态优先级

中央状态机负责状态竞争。总体优先级为用户直接互动、光标互动、AI 思考与说话、自主行为、普通待机。状态变化由 `PetController` 协调。

## 常见问题

### 填写 API Key 后仍无法聊天

检查 Base URL、模型名、网络连接与 `logs/maidie.log`。环境变量可能覆盖 JSON 中的主 AI Key。

### 倒计时被当成屏幕问题

当前 Router 已将 `time_now` 和 `time_delta` 与视觉意图区分。请确认运行的是最新代码，并运行 Router 与 TimeTool 测试。

### 屏幕理解没有运行

检查是否明确要求查看屏幕、视觉 Key 和 Workspace ID 是否配置，以及百炼模型权限。关闭 OCR 不会关闭按需 Qwen VL。

### 动作没有播放

确认素材存在于 `assets/actions/`，且文件名、帧数与 `actions.json` 一致；同时检查冷却和状态优先级。

### 联网搜索不可用

确认已启用联网查询并配置 Tavily Key。当前没有其他搜索 provider 的生产实现。
## 屏幕问题求解

- 明确要求看屏幕后，可识别代码报错、题目、界面问题和一般屏幕内容。
- Python 等技术报错在置信度足够时会生成精确查询，并按需补充搜索资料。
- 题目内容会保存在结构化上下文中，由回答模型给出思路和可执行答案。
- 搜索不可用时仍会依据已识别的屏幕事实回答，并明确说明资料补充失败。
- 普通聊天、模糊指代和低置信度视觉结果不会自动触发截图或盲目搜索。
