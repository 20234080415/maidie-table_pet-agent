# Roadmap

本文区分“已完成”“部分完成”和“计划中”，避免把基础接入写成成熟能力。

## 已完成

- Windows 透明置顶桌宠、拖拽缩放、围栏和数据驱动动作。
- 聊天气泡、流式文本、工具结果与长内容面板。
- `BrainRouter → BrainPlanner → BrainExecutor → ToolRegistry → Synthesizer` 生产链路。
- 时间、倒计时、天气、Tavily 搜索、SQLite 记忆和屏幕工具。
- Qwen VL 多范围截图、视觉会话追问、问题上下文与条件搜索。
- CodingAgentTool 的只读 OpenCode / Codex 接入、实时控制台和结构化分析卡片。
- 人格 Prompt、基础体验层、主动行为开关和 Windows 打包脚本。

## 部分完成 / 实验性

- LLM Router：已有结构化意图与规则降级，长尾表达仍依赖模型输出质量。
- Planner：常用工具链已覆盖，复杂多步计划和依赖恢复仍需加强。
- Memory：已有 SQLite 长期记忆与短期事件上下文，缺少完整管理、检索质量评估和用户可见控制。
- Vision：已有当前窗口、全屏、鼠标区域和框选；window-aware 上下文、OCR/Vision 协同与更多视觉任务仍有限。
- 系统工具：读取/搜索文件和截图为只读；创建文件、打开应用/目录、切换窗口和写剪贴板要求确认；任意 shell、删除和脚本执行未实现。
- 自动任务与提醒：调度器支持一次性、周期和条件任务的数据结构，但自然语言创建、查看、编辑、持久运行和完整 UI 闭环尚未完成。
- 主动行为和体验层：基础模块已接入，默认关闭，行为自然度与可解释控制仍需验证。

## 计划中

- 扩展并稳定 LLM Router、Planner 和复杂任务失败恢复。
- 增强 Vision API、window-aware Vision 和视觉隐私提示。
- 提供长期 Memory 的查看、删除、范围控制和质量评估。
- 丰富动画、情绪、注意力与低打扰行为编排。
- 为搜索、文件、视觉和任务提供更合适的 UI 面板。
- 建立更细粒度、可审计、可撤销的工具权限分级。
- 完成提醒/自动任务的自然语言和 UI 管理闭环。
- 提升 Router、Planner、工具、Qt 线程和真实 Windows 场景的测试覆盖。

Roadmap 不承诺版本或时间。功能只有在代码、权限边界、UI 状态和测试同时可验证后，才从“部分完成”移入“已完成”。
